Get the handle position and compute the vector between the two handles : handleVector.

Parameters
ttbPosePose2D object containing the TTB's pose

Both handles positions are computed. The desired one is chosen as being the closest. The vector handleVector is then computed as being the unit vector from the undesired handle to the desired one.

Warning
{Choosing the closest handle might not work in specific situations !}.
