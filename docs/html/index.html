2. Principles of the Navigation code.

v1.3 : using the command "rosrun navigation cmd_ouverture.py" launches the navigation code.

The file cmd_ouverture.py is the main file. It computes the desired trajectory based on the current stage and step using PointCreator and TrajFactory.

It then launches the control law in control.
